# 膜力佳台灣官方網站 MVP

這是手機、平板、電腦保護貼官方網站第一版，可直接部署到 GitHub Pages、Vercel 或 Cloudflare Pages。

## 目前內容

- 首頁 Hero
- 使用情境選膜
- 手機 / iPad / MacBook 產品線
- 產品卡片資料
- 選膜指南比較表
- 貼膜教學流程
- FAQ
- LINE 客服與企業採購 CTA

## 第一階段不包含

- 購物車
- 金流
- 會員登入
- 訂單管理
- 物流與電子發票

這些功能預留在第二階段商城開發。

## 部署到 GitHub Pages

1. 到 Settings → Pages
2. Source 選擇 `Deploy from a branch`
3. Branch 選 `main`，資料夾選 `/root`
4. 儲存後，網站會出現在：
   `https://leojung0823.github.io/molijia-tw-official/`

## 後續建議

第二階段建議改為 Next.js + 商品資料庫 + 金流物流串接，或直接接 Shopify / SHOPLINE。
